# Identificador de Chips 74 - Micro

Projeto acadêmico para identificação automática de chips da série 74xx utilizando Arduino Mega.
 Integrantes:
- Bruno Tardin
- Matheus Marques
- Thiago Bello

Sobre o Projeto:
O sistema identifica chips lógicos da série 74xx, exibindo o resultado em um display TFT 2.4" conectado ao Arduino Mega 2560. A interface web permite o gerenciamento dos chips cadastrados e as baterias de testes para sua identificação.

Progresso

### Meta 1 - 09/06/2026
- **Bruno Tardin** — Listagem e adição de chips na interface web
- **Matheus Marques** — Controle dos pinos de entrada e saída para 1 teste de 1 chip
- **Thiago Bello** — Tela de start e tela de chip identificado no display TFT

Tecnologias Utilizadas
- Arduino Mega 2560
- Display TFT 2.4" (ILI9341)
- Bibliotecas: MCUFRIEND_kbv, Adafruit GFX
